KONTAKTINFORMĀCIJA




Uzdevums: Izveidot web sistēmu, kura ļauj veidot uzņēmuma elektrisko velosipēdu lietošanas grafiku darbiniekiem (piesaistīt elektriskos velosipēdus darbiniekiem uz noteiktu laiku).

Prasības: 2 formas (viens saraksts ar darbiniekiem un elektriskajiem velosipēdiem no cikiem līdz cikiem kurš darbinieks lieto velosipēdu, darbinieku velosipēdu sasaistes forma)
- kontrole, lai datums/pulksteņa laiks nevar tikt ievadīts pagātnē;
- vienu velosipēdu var rezervēt vienlaicīgi tikai viens darbinieks;
- viens darbinieks var vienlaicīgi rezervēt vairākus velosipēdus;
- iespēja labot/dzēst ievadītos darbiniekus, velosipēdus.

Relāciju DB pēc paša izvēles, Uzdevums jārealizē primāri izmantojot Java;
Izdarīto iesniegt GitHub

Ieteicamās tehnoloģijas: Maven, Spring vai cits līdzvērtīgs web freimworks, SQLite vai MySQL.
